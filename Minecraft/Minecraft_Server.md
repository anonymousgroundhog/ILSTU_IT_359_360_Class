# Important details
- If trying to connect to the server from outside campus be sure to use a vpn to remotely connect to ISU's network.
- Additional details about this can be found at the following [link](https://help.illinoisstate.edu/technology/support-topics/network/vpn-at-illinois-state-university)
# Connecting to the Minecraft server

1. Launch Minecraft
2. Click on Multiplayer
3. Add Server
4. Server Name: ISU, Server Address: 10.110.10.150
