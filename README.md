# ILSTU_IT_359_360_Class
A repo for the IT 359 and IT 360 classes.

## Branch Details
- Each Branch naming scheme begins with `IT_[Class Number]_[Spring/Fall]_[Year]`
### Branch Layout
- Slides
  - Where class slides in PDF format are stored.
- Project
  - Project-related materials.
- In_Class_Activity_Files
  - All files and details for class day activities are stored.
- Minecraft
  - Details related to the Minecraft server.
